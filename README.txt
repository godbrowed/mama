ЩО МІНЯТИ

1. Встав свої фото в папку images.
2. Якщо хочеш не міняти назви в html, просто заміни файли з такими самими назвами:
   hero-main.jpg
   hero-small.jpg
   about.jpg
   product-1.jpg ... product-6.jpg
   gallery-1.jpg ... gallery-8.jpg

3. Тексти і ціни міняються в index.html.
4. Весь дизайн у css/style.css.
5. Скрипти в js/main.js.
