ЩО ЦЕ
Це вже не просто статичний сайт. Тут є сервер + Telegram бот.

ЯК ЗАПУСТИТИ
1. Встанови Node.js 18+
2. Відкрий папку mama1 в терміналі
3. Скопіюй .env.example в .env
4. Вкажи BOT_TOKEN бота від @BotFather
5. OWNER_USERNAME вже стоїть olenatimachova, якщо треба - зміни
6. Запусти:
   npm install
   npm start
7. Відкрий http://localhost:3000

ЯК ПІДКЛЮЧИТИ БОТА
1. Створи бота через @BotFather
2. Встав BOT_TOKEN у .env
3. Запусти сервер
4. Зайди в свого бота з аккаунта @olenatimachova
5. Напиши /start
Після цього бот запам'ятає chat_id і почне присилати нові замовлення.

ЩО ВМІЄ БОТ
- автоматично отримує нові замовлення з сайту
- має кнопки зміни статусу:
  Передано на виготовлення
  Очікує на доставку
  Передано на доставку
  Прибуло в пункт видачі
  Виконано
  Скасовано
- команди:
  /orders
  /order LC-1001
  /status LC-1001 shipping
  /cancel LC-1001

ЩО БАЧИТЬ КЛІЄНТ
- на сайті оформлює замовлення, нічого не пише в Telegram
- бачить повідомлення: менеджер зв'яжеться з вами
- у кошику бачить свої активні замовлення і статус
- може скасувати замовлення, поки воно не передано на доставку
